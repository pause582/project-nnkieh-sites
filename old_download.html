下載連結： ProjectNNKIEH Release1.0.zip (4.19MB)

如何安裝 Install

就如同一般的Minecraft存檔安裝方式。

  1. 下載地圖檔，解壓縮得到"Project NNKIEH"資料夾。
  2. 開啟Minecraft，本地圖檔不需要模組，原版即可。關於版本，見注意事項。
  3. 在主畫面中，選擇「選項...」 → 「資源包...」 → 「開啟資源包資料夾」，開啟這個資料夾。
  4. 回到這個資料夾的「上一層資料夾」，找到"saves"資料夾，將第一步的資料夾放入"saves"資料夾。
  5. 在Minecraft中選擇單人遊戲，應該就可以看見地圖了。

注意事項 Notice

  • 本地圖檔使用1.12.2版建造，您可以使用更高的版本，但本人無法其擔保相容性。
  • 本計畫圖片拍攝時使用光影模組，原版看到的景色可能與照片有差異。
  • 本計畫使用的光影模組是SEUS。安裝方式與連結在此恕不提供。
